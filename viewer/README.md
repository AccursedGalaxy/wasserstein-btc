# wbtc · viewer

A static research dashboard for the wbtc project. Renders the headline
numbers, cumulative CRPS curves, method comparisons, extended-baseline
panel, hyperparameter sweep, asset price + return charts, and full
run provenance — all from the existing `results/` and `data/`
directories.

The viewer is **read-only** and **does not modify any source code**.
It lives entirely in this directory.

## Layout

```
viewer/
  build_data.py   bundles results/ + data/ → public/data.json
  serve.py        tiny local HTTP server (stdlib only)
  public/
    index.html    page structure
    styles.css    modern dark/light theme
    app.js        ECharts + interaction (ES module, CDN-loaded)
    data.json     generated by build_data.py (gitignorable)
```

## Run it

```bash
# 1. bundle the data the page consumes
uv run python viewer/build_data.py

# 2. start the local server (opens your browser)
uv run python viewer/serve.py
```

Then click around at `http://127.0.0.1:8765/`. Re-run step 1 whenever you
add new results (e.g. after `wbtc backtest-long`).

## What's in it

| Section              | Source                                              |
|----------------------|------------------------------------------------------|
| Hero stats           | `results/MANIFEST.json` (most recent headline)       |
| Method cards         | hard-coded descriptions; method list from `data.json`|
| Headline table       | `results/MANIFEST.json`                              |
| Cumulative CRPS      | `results/long_<asset>_h<h>.json` (downsampled)       |
| Mean-CRPS + DM bars  | derived from the same JSON, DM computed in the build |
| Extended panel       | `results/extended_btc_h{1,5}.json`                   |
| Hyperparameter sweep | `results/hyperparam_sweep.csv`                       |
| Markets              | `data/<asset>_1d.parquet`                            |
| Provenance           | `results/MANIFEST.json` (latest entry)               |

## Notes

- Pure HTML/CSS/JS — no build step, no node_modules. ECharts is loaded
  from a CDN.
- `data.json` is ~2.8 MB after downsampling cumulative curves to 800
  points; load is one fetch, no streaming needed.
- The theme toggle in the topbar swaps light/dark CSS tokens and
  re-draws every chart.
- For a public deploy, you can just upload `viewer/public/` (plus a
  fresh `data.json`) to any static host (GitHub Pages, Netlify, etc.).
